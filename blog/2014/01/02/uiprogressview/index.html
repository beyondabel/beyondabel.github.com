iOS组件之UIProgressView详解

引导

该UIPickerView的类实现的对象，显示一个或多个值的集合。用户通过旋转轮选择值，这样的值所需的行对齐选择指标。UIProgressView是继承于UIView(有关UIView请参照《iOS组件之UIView详解》)。

目录

1、基本属性及方法
2、iOS5新增内容

1、基本属性及方法

// 初始化progressView的样式

- (id)initWithProgressViewStyle:(UIProgressViewStyle)style;

// progressView的样式（默认为UIProgressViewStyleDefault）

UIProgressViewStyle progressViewStyle;

// 进度（范围为0-1）

float progress;

2、iOS5新增内容

// 进度填充部分的颜色(譬如进度到40%,那么40%部分的颜色就是这个属性)

UIColor* progressTintColor;

// 未填充部分的颜色

UIColor* trackTintColor ;

// 进度填充部分的图片，如progressTintColor

UIImage* progressImage;

// 未填充部分的图片，如trackTintColor

UIImage* trackImage;

// 设置progress的进度

- (void)setProgress:(float)progress animated:(BOOL)animated;
